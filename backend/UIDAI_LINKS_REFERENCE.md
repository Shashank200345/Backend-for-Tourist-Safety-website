# UIDAI Links Reference - Correct URLs for Aadhaar Offline e-KYC

## 🔗 Official UIDAI Links

### Main Portal
**Primary URL:** https://resident.uidai.gov.in/

### Offline e-KYC Download Options

**Option 1: Through Main Portal (Recommended)**
1. Visit: https://resident.uidai.gov.in/
2. Login with Aadhaar number
3. Navigate to "Offline e-KYC" or "Paperless Offline e-KYC" section
4. Follow the steps to download

**Option 2: Direct Links (Try these if main portal doesn't work)**
- https://resident.uidai.gov.in/offlineaadhaar
- https://resident.uidai.gov.in/offline-kyc
- https://resident.uidai.gov.in/offline-ekyc

**Option 3: Search on UIDAI Website**
1. Visit: https://uidai.gov.in/
2. Search for "Offline e-KYC" or "Paperless Offline e-KYC"
3. Follow the link to resident portal

## 📋 Step-by-Step Process

### To Download Aadhaar Paperless Offline e-KYC:

1. **Access UIDAI Resident Portal**
   - Go to: https://resident.uidai.gov.in/
   - Or try direct link: https://resident.uidai.gov.in/offlineaadhaar

2. **Login**
   - Enter your 12-digit Aadhaar number
   - OR enter 16-digit Virtual ID (VID)
   - Click "Send OTP"
   - Enter OTP received on registered mobile number

3. **Generate Offline e-KYC**
   - Look for "Offline e-KYC" or "Paperless Offline e-KYC" option
   - Click on it

4. **Set Share Code**
   - Create a 4-character Share Code (password)
   - This encrypts the downloaded file
   - Remember this code - you'll need it to decrypt

5. **Download**
   - Click "Download" button
   - A ZIP file will be downloaded
   - Extract the ZIP file
   - Inside you'll find the XML file

6. **Extract XML**
   - The ZIP file contains the digitally signed XML
   - Extract and use the XML file in our application

## 🔍 Troubleshooting

### If Links Don't Work:

1. **Check UIDAI Website Status**
   - Visit: https://uidai.gov.in/
   - Check for any maintenance notices

2. **Try Different Browsers**
   - Chrome, Firefox, Edge
   - Clear browser cache

3. **Use Mobile App**
   - Download "mAadhaar" app from Play Store/App Store
   - Generate Offline e-KYC from the app

4. **Contact UIDAI Support**
   - Helpline: 1947
   - Email: help@uidai.gov.in

## 📱 Alternative: mAadhaar App

If the website doesn't work, you can use the official mAadhaar mobile app:

1. Download "mAadhaar" from Play Store/App Store
2. Login with Aadhaar number
3. Navigate to "Offline e-KYC" section
4. Generate and download the XML file

## ⚠️ Important Notes

- The XML file is encrypted with your Share Code
- Keep the Share Code safe - you'll need it to decrypt
- The XML file is digitally signed by UIDAI
- File is valid for a limited time (check expiry date)
- For testing, you can use the sample XML file provided in `test-frontend/test-aadhaar-sample.xml`
